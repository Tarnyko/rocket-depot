    Rocket Depot

    Rocket Depot is a simple graphical frontend for rdesktop and FreeRDP with support for connection profiles. It runs on Linux desktops using GTK and takes advantage of modern desktop environment features such as Unity Quicklists.

    Installation

    Ubuntu PPA 
    sudo apt-add-repository ppa:robled/rocket-depot
sudo apt-get update
sudo apt-get install rocket-depot
    PyPI 
    sudo pip install rocket-depot
